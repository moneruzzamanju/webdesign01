What is the way to be a full stack web developer?

A full stack web developer is a person who can develop both front end and back end website. He also knows how to work program in browser, server & database. Front end web developer should knows HTML, CSS, JavaScript, Bootstrap, jQuery, Angular/React/vue etc. Back end web developer should knows PHP/ ASP / C++ / C# / Java / Python / Node.js with a database e.g SQL/ SQLite/ MongoDB. Now a days, Wordpress or Laravel framework is compulsory for a web developer as Content management system (CMS). When a person knows front end web development & Back end web development , he will be a full stack web developer.
